### Filter Creation Help

1. Create a text file in ( ./data/filter )
    
    1. The file extension must be a txt 
    
2. Filter example 

   ```
   - filename: test.txt
   
   cat, meow
   miaow
   ```
   - result: ["cat", "meow", "miaow"]

3. Add "," between words to finish!

_ _ _ _ _
### 필터 작성 도움말

1. 텍스트 파일을 ( ./data/filter )에 생성하기
    
    1. 파일의 확장자는 txt 이여야 합니다
    
2. 필터 예시 

   ```
   - 파일명: test.txt
      
   고양이, 야옹
   애옹
   ```
   - 결과: ["고양이", "야옹", "애옹"]

3. ","를 단어 사이에 추가하면 끝!
